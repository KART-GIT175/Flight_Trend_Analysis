# Flight Passenger Trend Analysis

## Description

This project explores historical airline passenger data from 1949 to 1960 using Python, Pandas, Seaborn, and Matplotlib. The objective of the analysis is to identify long-term growth trends, seasonal travel patterns, and monthly fluctuations in passenger traffic.

Through exploratory data analysis (EDA), the project examines how passenger volumes changed over the years and identifies peak travel seasons. Various visualizations, including line plots, bar charts, and heatmaps, are used to uncover meaningful insights and present data in an intuitive manner.

The analysis reveals a significant increase in air travel demand throughout the observed period, indicating the rapid growth of the aviation industry. Seasonal patterns were also identified, with certain months consistently recording higher passenger traffic than others.

### Technologies Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib

### Key Objectives

* Analyze year-wise passenger growth.
* Identify peak and low-demand travel months.
* Explore seasonal travel behavior.
* Visualize trends using statistical charts.
* Generate business insights from historical data.

### Key Findings

* Passenger traffic increased steadily from 1949 to 1960.
* Mid-year months showed the highest travel demand.
* Clear seasonal patterns were observed across multiple years.
* Air travel experienced strong and consistent long-term growth.

This project demonstrates practical data analytics skills, including data exploration, visualization, trend analysis, and insight generation using real-world time-series data.
